# LPC2148

Example Code LPC2148

This Source is having following libraries.
	
	1. GPIO
	2. Serial
	3. ADC
	4. I2C
	5. SPI
	6. RTC
	7. Interrupt
	8. Delay
	9. String library

Following Sensor and Modules are interfaced using those libraries.
	
	1. DHT11
	2. GSM 
	3. Ultrasonic sensor ( HCSR-04 )
	4. LCD 16x2
	5. MCP3204 ( 12 bit ADC )
	6. EEPROM  ( 24LC02 )
	7. RTC 	   ( DS 1307 )
	8. OLED Display ( SSD1306 )
	9. Keypad ( 4x4 )
